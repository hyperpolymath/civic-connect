== Current State

Civic-Connect is a privacy-first civic engagement platform with three layers:
IndieWeb2 Bastion (hardened ingress), Civic-Stream (government API integration),
and CivicConnect Vision (gamified participation, planning stage).

Key capabilities delivered:

* IndieWeb2 Bastion:
** GraphQL DNS API (Rust/Axum, port 8443) -- 70% complete
** Oblivious DNS (Rust/Tokio, port 853/8853)
** Consent API (Deno, port 443)
** Policy Gate (ReScript + Deno, 9 validators)
** 13 crypto policy requirements (CPR-001 through CPR-013)
** Post-quantum crypto: Kyber-1024 KEM, Dilithium5 signatures, SPHINCS+ fallback
** Hybrid signatures: Ed448 + Dilithium5 dual-sign
** Zero-knowledge DNS (server never sees plaintext queries)
* Civic-Stream: government API integration (40% complete)
* SurrealDB provenance graph
* Stapeln container deployment (cerro-torre build/sign, selur-compose)
